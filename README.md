# 扱うデータ（予定）
収集し扱う予定のデータは以下の通り。

## 雑誌連載分
- 作品名
- 作者名
- 掲載雑誌
- 掲載順（平均と推移）
- センターカラー数
- 連載期間
- 総話数

## 単行本分
- 作品名
- 作者名
- 巻数
- 発売日
- ISBN
- センターカラー数
- KR menu
- アマゾンのリンク
